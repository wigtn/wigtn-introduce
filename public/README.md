# Public Assets

## Video Background

The Hero section expects a video file at `/placeholder-video.mp4`.

### Options:

1. **Use a stock video**: Download a tech/abstract video from:
   - https://www.pexels.com/videos/
   - https://pixabay.com/videos/
   - https://www.videvo.net/

2. **Remove video and use gradient only**: The component already has a fallback gradient animation

3. **Replace with animated SVG**: Create a custom animated background

### Recommended Search Terms:
- "technology abstract"
- "digital network"
- "particles animation"
- "data flow"
- "futuristic background"

Place your video file as: `public/placeholder-video.mp4`
